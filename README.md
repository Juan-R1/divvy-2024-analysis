# Divvy Bike Share Analysis 2024

A data analytics portfolio project exploring Chicago Divvy Bike Share trips to identify rider behavior, usage patterns, seasonal trends, and operational insights.

## Project Status

Status: Portfolio analytics project with a notebook and exported HTML analysis.

## Business Problem

Bike-share operators need to understand how rider segments behave across time, season, and location. These insights can support decisions around station planning, rider engagement, and operational planning.

## Key Questions

- How do casual riders and member riders differ in usage behavior?
- When do riders use the system most by hour, day, and season?
- Which stations or time periods show higher demand?
- What recommendations can be made from the data?

## Key Findings

- Member riders show stronger weekday usage patterns.
- Casual riders show stronger weekend and warmer-month usage patterns.
- Ride volume rises during summer and declines during colder months.
- Station and timing patterns can inform planning and reporting.
